## ğŸ¨ Customization

### Update Personal Information
Edit the content files in `/data` or `/content` directory:
- `about.json` - Personal information
- `projects.json` - Project details
- `experience.json` - Work history
- `skills.json` - Technical skills

### Modify Theme Colors
Update the color scheme in `tailwind.config.js` or your CSS files to match your preference while maintaining the XP aesthetic.

### Add New Applications
Create new window components in `/components/windows/` and register them in the desktop configuration.
